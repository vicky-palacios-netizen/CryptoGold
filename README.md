Segunda Evaluación - Programación III

Billetera de criptomonedas desarrollada con ASP.NET Core, SQL Server y Vue.js. (CryptoGold)

Funcionalidades
Alta de compra de criptomonedas.
Consulta de cotización mediante CriptoYa.
Historial de movimientos.
Almacenamiento de datos en SQL Server.
Proyectos
CryptoAPI (Backend)
CryptoVue (Frontend)
Ejecución
Backend

Abrir la solución ubicada en la carpeta CryptoAPI y ejecutar el proyecto desde Visual Studio 2022.

Frontend

Abrir la carpeta CryptoVue y ejecutar:

npm install

npm run dev

Luego abrir la dirección indicada por Vite en el navegador.
