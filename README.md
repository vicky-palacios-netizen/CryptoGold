Segunda Evaluación - Programación III

CryptoGold es una billetera de criptomonedas desarrollada utilizando ASP.NET Core, SQL Server y Vue.js. El sistema permite registrar compras de criptomonedas, consultar cotizaciones mediante la API CriptoYa y visualizar el historial de movimientos almacenados en la base de datos.

El proyecto está compuesto por dos partes: CryptoAPI (backend) y CryptoVue (frontend).

Para ejecutar el proyecto, abrir la solución ubicada en la carpeta CryptoAPI y ejecutar la aplicación desde Visual Studio 2022. Luego abrir la carpeta CryptoVue y ejecutar los comandos:

npm install

npm run dev

Una vez iniciado el frontend, abrir en el navegador la dirección indicada por Vite.
